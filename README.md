# Hostel Management System

A web-based Hostel Management System for managing students, rooms, QR-based IN/OUT attendance, complaints, and warden/student dashboards.

## Features

- Warden Login
- Student Login
- Student Management
- Room Allocation and Deallocation
- Dynamic Hostel Room View
- QR Attendance System
- Manual Attendance Backup
- Attendance History
- Complaint Management
- Student Profile
- Change Password
- System Overview Dashboard

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP Apache Server
- QRCode.js
- html5-qrcode
